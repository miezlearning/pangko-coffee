JSON Excel Manual

Form Import

Pilih file, tetapkan metode default (opsional), lalu klik Import.

Dukungan: .json atau .xlsx (Excel). Gunakan template di kanan untuk struktur kolom.

Template & Contoh

Lihat contoh struktur data terlebih dahulu, lalu unduh template.

  • Kolom minimal: orderId, userId, customerName, status, createdAt, items (JSON).
  • items berbentuk array objek: { id, name, price, quantity, notes }.
  • createdAt dapat berbentuk ISO string atau tanggal Excel.

Tips

  • Import mempertahankan tanggal asli untuk akurasi historis.
  • Jika ingin muncul di metrik hari/bulan ini, ubah createdAt sebelum upload.
  • Status yang mempengaruhi revenue: paid/processing/ready/completed.
